Palavras a serem encontradas:

    Selecione as palavras na grade!

    Como Jogar

    1. Use o mouse para selecionar as palavras que encontrar

    2. As palavras podem estar em qualquer direção:

    • Horizontal (→ ←)
    • Vertical (↑ ↓)
    • Diagonal (↘ ↙ ↗ ↖)

    3. Quando encontrar uma palavra, ela será destacada em verde

    4. Encontre todas as palavras para vencer!
